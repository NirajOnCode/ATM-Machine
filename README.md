#  ATM Machine Simulator 

A console-based ATM simulation project created using *Java (OOPs Concept)*.  
This project demonstrates the use of *classes, objects, inheritance, and encapsulation* in a real-world scenario.

It simulates basic ATM operations like balance checking, withdrawal, and deposit through user input.

---

##  Features
-  Check account balance.  
-  Withdraw or deposit money.  
-  Secure PIN-based access.  
-  Demonstrates core *Object-Oriented Programming* concepts.
